# 高爾夫預約系統 — 教練使用說明

---

## 一、審核預約

學員預約後，你會收到通知。回覆：

- **核准** `核准 單號`  例：核准 ORD-XXXXXX
- **拒絕** `拒絕 單號`  例：拒絕 ORD-XXXXXX

---

## 二、查詢與排程

| 指令 | 說明 |
|------|------|
| 查詢今日 | 今天所有預約 |
| 查詢明日 | 明天所有預約 |
| 查詢 3/12-3/15 | 指定日期區間的預約 |
| 排程今日 | 今天排程（精簡版） |
| 排程明日 | 明天排程（精簡版） |
| 排程 3/12 | 指定日期的排程 |

---

## 三、核銷堂數

**方式一：依訂單**

```
完成 單號
```
例：`完成 ORD-XXXXXX`

**方式二：依姓名 + 日期**

```
姓名 日期 完成(N)
```
例：`sam kuo 3/10 完成(1)` 扣 1 堂  
例：`sam kuo 3/10 完成(3)` 扣 3 堂  

核銷後會自動通知學員剩餘堂數。

---

## 四、開課（儲值課程）

學員付款後，為他開通堂數：

```
開課 學員姓名 方案名稱 堂數
```
例：`開課 王小明 10堂一對一 10`

學員姓名會自動對應 LINE 顯示名稱。開通後學員需輸入「確認課程」才會啟用。

---

## 五、其他

| 指令 | 說明 |
|------|------|
| 管理取消 單號 理由 | 直接取消預約（無 2 小時限制） |
| 查詢學員 姓名 | 查詢學員的 LINE UID |
| 使用教學 | 查看本說明 |

---

## 六、自動提醒

- 每日 **21:00** 自動提醒學生明天有課
- 每日 **21:00** 推播教練明日行程

---

*輸入「使用教學」可隨時在 LINE 查看精簡版說明*
