# 高爾夫預約系統 — 測試清單

> 開始測試前，請確認：
> - [ ] LINE Webhook URL 已設為 `https://golf-coach-api-637405580782.asia-east1.run.app/webhook`
> - [ ] LIFF Endpoint URL 已設為 `https://golf-coach-aebcd.web.app/coach/index.html`（或您部署的網址）
> - [ ] 您已將 Bot 加為 LINE 好友

---

## 一、教練 LINE 訊息測試（用您的 LINE 帳號）

| 測試項目 | 輸入訊息 | 預期結果 |
|----------|----------|----------|
| 使用教學 | `使用教學` 或 `help` | 回覆教練指令說明 |
| 查詢今日 | `查詢今日` | 回覆今日預約（可能為空） |
| 查詢明日 | `查詢明日` | 回覆明日預約（可能為空） |

---

## 二、教練 LIFF 頁面測試

1. **開啟 LIFF**  
   - 在 LINE 中點擊 LIFF 連結，或從 LIFF 設定的 Endpoint URL 直接開啟  
   - 應會要求 LINE 登入，登入後進入教練後台

2. **休假設定**（可獨立測試）  
   - 切換到「休假設定」分頁  
   - 選擇日期、填寫備註，點「新增休假」  
   - 應顯示「已新增」，下方出現剛新增的休假

3. **扣課 / 加課**（需先有學員）  
   - 學員下拉選單：若尚無學員，會是空的  
   - 需先用另一個 LINE 帳號加 Bot 好友，才會出現在學員名單  
   - 選學員 → 輸入堂數（正數加課、負數扣課）→ 送出

4. **學員明細**  
   - 切換到「學員明細」分頁  
   - 顯示所有曾加 Bot 好友的學員（含別名、堂數）

---

## 三、學員 LINE 訊息測試（用另一個 LINE 帳號）

1. **加 Bot 為好友**  
   - 用第二個 LINE 帳號掃描 Bot 的 QR Code 加好友

2. **使用教學**  
   - 輸入 `使用教學`  
   - 應回覆學員指令說明

3. **我的堂數**  
   - 輸入 `我的堂數`  
   - 若尚未加課，應顯示「您目前沒有可用的儲值課程包」

4. **查詢預約**  
   - 輸入 `查詢預約`  
   - 若無預約，應顯示「您目前沒有有效預約」

---

## 四、完整流程測試（教練 + 學員）

1. **學員**：用第二個 LINE 帳號加 Bot 好友  
2. **教練**：在 LIFF 選該學員，輸入 `5`（加課 5 堂），送出  
3. **學員**：傳 `我的堂數` 給 Bot  
   - 應顯示剩餘 5 堂  
4. **教練**：在 LIFF 選該學員，輸入 `-1`（扣課 1 堂），送出  
5. **學員**：再傳 `我的堂數`  
   - 應顯示剩餘 4 堂  

---

## 五、若測試失敗，可檢查

| 現象 | 可能原因 |
|------|----------|
| 傳訊息給 Bot 沒反應 | Webhook URL 未設定或錯誤；Cloud Run 服務未運行 |
| LIFF 顯示「Invalid LIFF ID」 | LIFF ID 或 Endpoint URL 設定錯誤 |
| LIFF 學員下拉選單為空 | 尚無學員加 Bot 好友；或 API 回傳 403（教練身份未通過） |
| 加課/扣課失敗 | 學員無有效課程包；或 API 錯誤（可看 Cloud Run Logs） |
| 403 Forbidden | 您的 LINE userId 未在 COACH_LINE_USER_IDS 中 |

---

## 六、查看 Cloud Run 日誌

若出現錯誤，可到 [Cloud Run Console](https://console.cloud.google.com/run) → 點選 `golf-coach-api` →「日誌」查看詳細錯誤訊息。
